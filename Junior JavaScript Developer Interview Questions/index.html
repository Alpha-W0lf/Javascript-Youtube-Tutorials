What are the data types supported by JavaScript?

primitive (numbers, booleans, strings, symbols, null, undefined), objects

python: numbers (integers, floats, complex), strings, booleans, arrays, tuples, lists, dictionaries

What do break and continue statements do?

break: jumps out of a loop

continue: jumps over this iteration of a loop

What's the difference between var, let, and const?

var: declares a variable. scope is usually global. can be used locally/within a function or block of code

let: usually used within a function or block of code (now preferred over var even globally since it solves the problem of redeclaration inside a block of code continuing after and outside the block)

const: similar to let, but immutable. cannot later be changed and can only be accessed within the block it was declared. short for "constant"

Explain the difference between null, undefined, and undeclared variables

null: variable is assigned "empty". null is a legitimate JS data type

undefined: a variable can be declared, but not have an assigned value. this would be undefined

undeclared: a variable must be declared in JS. if it hasn't yet been declared above, the program will have nothing to reference

What is the difference between == and ===?

abstract comparison. == equates type before answering. This is called type coercion. Ex: would return 2 == "2" as true since it allows int and string data types to be compared

strict comparison. === does not equate type before answering. Does not perform type coercion. Ex: would return 2 === "2" as false since it does not allow int and string data types to be compared
